# Poultry Vision Monitoring

Computer vision system for intelligent poultry monitoring.

## Project context

This project is developed as part of the course:

Proxecto integrador de Intelixencia Artificial I (PIIA1)  
Bachelor's Degree in Artificial Intelligence  
Universidade de Santiago de Compostela

## Objective

The goal of the project is to explore computer vision techniques to address several use cases related to poultry monitoring.

## Repository structure


data/
  01_raw     -> original datasets
  02_work    -> prepared datasets
  03_final   -> final dataset for training

scripts/
  dataset/   -> dataset preparation and analysis scripts

docs/
  datasets.md -> dataset documentation
